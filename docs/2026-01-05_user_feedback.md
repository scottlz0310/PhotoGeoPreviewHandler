- Optional [88]（ブロック済みの実行可能ファイル）はオンライン審査の通過実績があるため、現時点は改修しない。

## 審査提出用コメント案（Optional [88]）

Optional [88] の指摘は、アプリ内から外部ブラウザを開く導線（`Launcher.LaunchUriAsync`）が原因と考えられます。アプリは外部実行ファイルを同梱・起動しておらず、ユーザー操作により外部 URL を開く用途のみです。Required の指摘が出た場合に限り改修を検討します。

## Microsoft Store 提出メモ

### 依存関係の説明文（追記案）

このアプリを実行するには .NET Desktop Runtime が必要です。Microsoft Store からのインストール時に自動的に提供されます。

### 説明文の整理案（短文）

写真の位置情報を地図で確認できる Windows アプリです。フォルダ内の写真を一覧表示し、EXIF/GPS を読み取って地図上にピンを表示します。撮影日時やカメラ情報も確認できます。

### キーワード候補（Store 検索向け）

photo, geotag, EXIF, GPS, map, location, photography, picture, map viewer, metadata

### スクリーンショット見直しポイント

- フォルダ一覧 + プレビュー + 地図の 3 ペインが一目で分かる構図
- 位置情報あり/なしの両方が分かる画面を含める
- ファイル関連付けの導線が分かる画面は 1 枚で十分
