# 手動テストチェックリスト

日常の品質確認は手動テストを重視します。E2E は UIA(FlaUI) で実アプリを起動するため
環境依存の不安定さがあり、通常はスキップします。

## 実行前

- `dotnet build PhotoGeoExplorer.sln -c Release -p:Platform=x64`
- `dotnet run --project PhotoGeoExplorer/PhotoGeoExplorer.csproj -c Release -p:Platform=x64`

## 基本動作

- アプリが起動し、クラッシュしない
- フォルダ選択ができ、一覧に画像が表示される
- 画像プレビューが表示できる
- 地図が表示され、ピンが表示される

## EXIF 編集

- JPEG 1 枚選択で「EXIF編集...」が有効
- 複数選択/フォルダ選択で EXIF 編集が無効またはエラーメッセージ
- 撮影日時のチェックをオフにすると日時UIと「更新日時同期」が無効
- 撮影日時のチェックをオンにすると日時UIと「更新日時同期」が有効
- 位置情報を入力して保存すると地図のピンが更新される
- 「地図から位置情報を取得」→地図クリックで緯度/経度が反映される
- 位置選択中に右クリックでキャンセルできる
- 位置選択中にドラッグで地図をパンできる
- 位置情報クリアで地図のピンが消える
- 撮影日時を更新しない場合に既存の日時が維持される
- 「更新日時同期」オンの場合、ファイル更新日時が撮影日時に一致する

## 期待されるログ
