PhotoGeoExplorer Detailed Help

A guide to reviewing and organizing photo locations on a map.

Overview Requirements & Formats Getting Started Screen Layout Navigation File Operations Search & View Preview Map Multi-Select EXIF Editing Settings Logs/Updates/Support Privacy

Overview

PhotoGeoExplorer displays photo location data (EXIF/GPS) on a map to help you review and organize shooting locations.

  • Review the photo list, preview, and map in a single screen.
  • Check shooting date/time and camera info from map pins.
  • Edit EXIF data when needed (JPEG only).

Requirements & Formats

System requirements

  • Windows 10/11
  • WebView2 Runtime (for displaying detailed help)

Supported image formats

  • .jpg / .jpeg / .png / .gif / .bmp / .tif / .tiff / .heic / .webp

File associations

  • File associations are supported for .jpg / .jpeg / .png / .heic.
  • When you open an associated image, the app loads its folder automatically.

Getting Started

  • Select a folder from [File] > [Open folder...] to load photos.
  • You can also drag and drop a folder into the app.
  • Select a photo to show the preview and map on the right.
  • Map pins appear only for photos that contain location data.
  • The last opened folder is restored on startup (if missing, the closest parent folder is used).

Screen Layout

  • Left File Browser: navigate folders, search, and view items.
  • Right Top Preview: inspect the selected photo.
  • Right Bottom Map: view and interact with location pins.
  • Bottom Status Bar: item counts, selection, and GPS status.

Navigation

  • Use Back/Forward to move through recent folder history.
  • Use Up and Home for quick navigation.
  • Click the breadcrumb separator ">" to open child levels.

File Operations

  • Create new folders, rename, move, and delete items.
  • Drag and drop files or folders to move them.
  • Double-click a folder to open it.

Search & View

  • Filter by file name using the search box.
  • Use Filters or View > Show images only to narrow results.
  • Change view mode (Icon/List/Details) from the View menu.
  • Details view shows extra information on hover.

Preview

  • Use the preview buttons to zoom, fit, or maximize.
  • Use previous/next to move between photos.
  • When maximized, the map area is adjusted automatically.

Map

  • Drag to pan and use the mouse wheel to zoom.
  • Click pins to view shooting time, camera, file name, and resolution.
  • Clicking a pin selects the corresponding photo in the list.
  • Open the location in Google Maps from the pin details.
  • Switch map tiles between OpenStreetMap and Esri WorldImagery (satellite imagery).
  • Map tiles are cached locally for faster reloading.

Multi-Select & Rectangle Selection

  • Use Ctrl/Shift for multi-selection.
  • Hold Ctrl and drag on the map to create a selection rectangle.
  • Photos inside the rectangle are selected and reflected in the list.

EXIF Editing

  • Right-click a photo and choose Edit EXIF (JPEG only).
  • Edit shooting date/time, latitude, and longitude.
  • Use "Get location from map" to pick a location by clicking the map.
  • Right-click cancels location selection.

Settings

  • Switch language and theme (restart required after change).
  • Change the default map zoom level and tile source.
  • Export and import settings.

Logs/Updates/Support

Logs

  • Open logs from Help > Open Log Folder.
  • Log location: %LocalAppData%\\PhotoGeoExplorer\\Logs\\app.log

Updates

  • Check updates via Help > Check for updates (GitHub Releases).

Support

  • Report issues on GitHub Issues.
  • Include repro steps and logs when possible.

Privacy

  • Photos and settings are processed locally and not uploaded.
  • Map tiles are fetched from OpenStreetMap and Esri servers.
  • See the privacy policy for details.
